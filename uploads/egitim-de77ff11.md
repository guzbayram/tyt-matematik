## 🧠 PEDAGOJİK MİMARİ

### Öğrenme Döngüsü (Her Konu İçin)
```
KEŞİF → ÖZET → GÖRSEL → ÖRNEK → PRATIK → PUAN
  ↓        ↓       ↓        ↓        ↓       ↓
Merak   Kural   Animasyon  Çözüm   Quiz   Rozet
```

### Mikro-Öğrenme Prensipleri
- **Maksimum 3 kural** per ekran (bellek yükü azaltma)
- **Her kural → 1 örnek** (somutlaştırma)
- **Her konu → 5 soru minimum** (pekiştirme)
- **Hata analizi:** "Yanlış yaptın, işte neden…" açıklaması
- **Zorluk artışı:** Kolay → Orta → Zor (scaffolding)


```
  │    ├── .example-panel    (çözümlü örnek)
  │    └── .quiz-panel       (alıştırma soruları)

  ├── DATA: topicsData[]     (125 konu verisi)
```

## 🔄 ALGORITMA: Öneri Sistemi

```javascript
function getRecommendedTopic(userProgress) {
  // 1. Hiç başlanmamış, düşük zorluklu konular öncelikli
  // 2. Zayıf alanlardaki konular (quiz skoru < 60%)
  // 3. Önceki ünite tamamlanmışsa yeni ünite açılır
  // 4. Günlük hedef: 3 yeni konu veya 2 tekrar
  
  const incomplete = topics.filter(t => !progress.completed.includes(t.id));
  const weak = topics.filter(t => (progress.scores[t.id] || 0) < 60);
  
  // Zayıf konuları tekrara sok
  if (weak.length > 0) return { type: "review", topic: weak[0] };
  
  // Yeni konu öner (zorluk sırasına göre)
  return { type: "new", topic: incomplete.sort((a,b) => a.difficulty - b.difficulty)[0] };
}
```

## 📎 EK: PDF'den Çıkarılan Kritik Formüller

### Sayılar Ünitesi
- Ardışık sayı: `n = (son-ilk)/fark + 1`
- Ardışık toplam: `S = (son+ilk)/2 × n`
- Bölme: `A = B·C + K`, `0 ≤ K < B`
- EBOB × EKOK = A × B

### Cebir Ünitesi
- Mutlak değer: `|a| = a (a≥0), -a (a<0)`
- Üs kuralları: `aⁿ·aᵐ = aⁿ⁺ᵐ`, `(aⁿ)ᵐ = aⁿᵐ`
- Kök: `ⁿ√a · ⁿ√b = ⁿ√(a·b)`
- Diskriminant: `Δ = b²-4ac`
- Vieta: `x₁+x₂ = -b/a`, `x₁·x₂ = c/a`

### Geometri Ünitesi
- Pisagor: `a² + b² = c²`
- Öklid: `h² = p·k`, `c² = p·a`, `b² = k·a`
- Üçgen alanı: `A = (a·h)/2 = (a·b·sinC)/2`
- 30-60-90: kenarlar `1 : √3 : 2`
- 45-45-90: kenarlar `1 : 1 : √2`
- sin30°=1/2, cos30°=√3/2, tan30°=√3/3
- sin45°=cos45°=√2/2, tan45°=1
- sin60°=√3/2, cos60°=1/2, tan60°=√3
